FUEL CALCULATOR

Este proyecto nace con la necesidad de emplear una buena planificación para carreras endurance.

Como SimRacers necesitamos una buena estrategia y planificación para sacar la mayor ventaja posible en las carrera, las cuales poseen una duración de 3 a 24 horas.

En esta primer versión solo se incorpora un "Fuel Calculator", lo cual nos permite calcular de forma fácil, segura y rápida el combustible que necesitaremos para finalizar la carrera.

Usando el combustible justo y necesario, permitida exprimir esas décimas que puedan marcar una diferencia a lo largo de la carrera.
